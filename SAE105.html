// 03 — English summary

Project overview

English summary

SAÉ 1.05 was a software development project aimed at building a modular tool for identifying and safely removing large files on a server. The solution consisted of three components: a PowerShell orchestrator (sae.ps1) handling environment setup and network drive mounting, a Python engine (analyse.py) performing recursive file system traversal using os.walk and exporting results as JSON, and a PyQt5 graphical interface (main_gui.py) for visualizing file sizes, selecting files, and triggering secure deletion.

This project developed my skills in Python programming, inter-process communication, PowerShell scripting, GUI development with Qt, and modular software architecture — all core competencies of the RT3 track (Creating R&T Tools and Applications) in the BUT Réseaux & Télécommunications program.
